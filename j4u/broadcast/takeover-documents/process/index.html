            ↘ PTZ ↗                                                                                ↘ 빔프로젝트 → 현장 예배자

종류

이모티콘

설명

카메라

카메라 → 컨트롤러 → PC → OBS → Youtube → 온라인 예배자

 - 일반카메라: Main(메인), Side(사이드), Piano(피아노) 카메라. 총 3대

 - PTZ 카메라(pan–tilt–zoom camera): 예배당 뒤에 고정 되어있는 카메라. 총 2대

PTZ

PTZ 컨트롤러 ↔ (PTZ)카메라

 - PTZ 카메라를 제어해주는 컨트롤러

 - 화면 컨트롤러에서(1, 2)

컨트롤러

(3,5,6)

(6→5)

(5 fix)

(6→5 cut)

(3↔5)

카메라 → 컨트롤러 → PC → OBS → Youtube → 온라인 예배자

 - 카메라에서 들어오는 화면을 전환을 조정해주는 역할

 - 컨트롤러 패드 구성

   - 1(PTZ 1번 예배당 뒤 정 가운데)

   - 2(PTZ 2번 예배당 뒤 오른쪽)

   - 3(PC화면/Piano 카메라),

   - 5(Main 카메라),

   - 6(Side카메라)

 - 자막 영상을 영상 위에 나오도록 제어 가능 DSK1

 - 패드 1번의 경우 다른 컨트롤러로 연결 (PC화면/Piano 카메라)

 - 3,5,6: 3,5,6 번갈아 가면서 적절히 전s환

 - 6→5: Side(6)전환 후 Main(5) 전환

 - 5 fix: Main(5)로 고정

 - 6→5 cut: Side(6)에서 Main(5)으로 cut 전환

 - 3↔5: Piano로 시작 후 Main, Side 번갈아 가면서 전환 후 마지막은 Main

PC

카메라 → 컨트롤러 → PC → OBS → Youtube → 온라인 예배자

 - Youtube 송출, OBS 관리, 영상 녹화, 광고 영상 등을 담당하는 중요한 컴퓨터

OBS

카메라 → 컨트롤러 → PC → OBS → Youtube → 온라인 예배자

 - PC로 들어오는 영상(Input)을 Youtube(Output)으로 보내주는 프로그램

 - Youtube로 영상 송출, 영상 녹화 중지 등을 담당

Youtube

카메라 → 컨트롤러 → PC → OBS → Youtube → 온라인 예배자

 - OBS에서 받은 영상을 실시간 스트리밍이 가능하게 해주는 Youtube

 - 썸네일, 설명, 실시간 추이 확인, 영상 송출 확인  

무전기

예배 진행시 예배 관계자들이 상황을 주고받기 위해 사용

 - 중계자 ↔ 예배 MC  

보이스톡

예배 진행시 카메라 분들과 상황을 주고 받기 위해 사용

 - 중계자 ↔ Main 카메라, Side 카메라
